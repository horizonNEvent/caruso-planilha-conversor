# Conversor de Planilha Caruso (Excel para TXT)

Este é um aplicativo Streamlit para converter planilhas Excel em arquivos TXT de largura fixa.

## Como usar
1. Faça o upload do arquivo `.xlsx`.
2. O script lerá a aba `preencher`.
3. Clique em converter e baixe o resultado.

Desenvolvido para automação de processos.
